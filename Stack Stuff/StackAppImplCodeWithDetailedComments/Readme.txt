Folder stack_app:
*Code for how to use the predefined stack data structures in the Java API 
*  interfaces: Deque
*  classes: Stack, ArrayDeque (used as stack), LinkedList class (used as stack)
*They are placed in the package stack_app

Folder stack_impl:
*Code for how to implement the stack data structure using different data structures for the stack content
*  Array, array list, linked list
*They are placed in the package stack_impl
